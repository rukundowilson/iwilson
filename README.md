# Wilson's Portfolio Website

A modern, responsive portfolio website built with HTML5, JavaScript, and Tailwind CSS.

## Features

- **Responsive Design**: Works perfectly on all devices (desktop, tablet, mobile)
- **Modern UI**: Clean and professional design with smooth animations
- **Interactive Elements**: 
  - Smooth scrolling navigation
  - Accordion sections for skills
  - Animated project cards
  - Working contact form with validation
- **Portfolio Showcase**: Display of key projects with descriptions and tech stacks
- **About Section**: Personal bio and skills overview
- **Contact Form**: Functional form with client-side validation

## Technologies Used

- **HTML5**: Semantic markup
- **JavaScript**: Interactive functionality
- **Tailwind CSS**: Utility-first styling via CDN
- **Font Awesome**: Icons

## Getting Started

Simply open `index.html` in your web browser. No build process or dependencies required!

## Customization

To customize the portfolio:

1. **Personal Information**: Update the hero section in `index.html`
2. **Projects**: Modify the project cards in the portfolio section
3. **Skills**: Edit the Languages, Frameworks, and Skills sections
4. **Images**: Replace placeholder images with your own
5. **Social Links**: Update social media links throughout the page

## Contact Form

The contact form includes:
- Name field validation
- Email format validation
- Message field validation
- Success/error message display

Note: The form currently uses client-side validation. To make it fully functional, you'll need to integrate it with a backend service or email API.
